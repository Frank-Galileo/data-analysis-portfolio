# data-analysis-portfolio
# Analyse de données - projet Kaggle

Ce projet présente une analyse réalisée sur Kaggle dans le cadre de mon portfolio data.

## Objectif
Analyser les données pour identifier des tendances, produire des visualisations et formuler des recommandations business.

## Outils
- Python
- Pandas
- NumPy
- Matplotlib / Seaborn

## Fichier principal
- `brazilian-e-commerce-marketplace-analysis.ipynb`

**1. Project Overview**
This project analyzes the Olist e-commerce dataset in order to understand
sales performance, customer satisfaction, and marketplace trends.

The objective is to extract actionable business insights from the data
and provide recommendations to improve marketplace performance.

**2. Dataset**
The dataset used in this project comes from the Olist Brazilian
e-commerce marketplace.

It contains information about:
  - customers
  - orders
  - payments
  - products
  - reviews
  - sellers

**3. Tools Used**
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

**4. Key Analysis**
The analysis focused on several business questions:
  1. Which product categories generate the most revenue?
  2. How does delivery performance affect customer satisfaction?
  3. Which regions generate the highest sales?

**5. Insights**
- Health & Beauty and Watches are among the most profitable categories.
- Faster deliveries tend to generate higher review scores.
- São Paulo represents the largest share of marketplace revenue.

**6. Business Recommendations**
Based on the analysis, several opportunities can improve marketplace performance:

- Improve logistics and delivery performance
- Focus marketing on high-performing product categories
- Strengthen presence in high-revenue regions
